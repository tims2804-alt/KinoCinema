## 3. Вывод о проведенном тестировании

В ходе выполнения работы были успешно протестированы программные модули приложения. Все запущенные тесты прошли успешно (статус: **Passed**).

### Причины успешного выполнения тестов:

* **Позитивные сценарии:** Система корректно предоставляет доступ пользователям, чьи данные (Логин/Пароль) совпадают с записями в БД, и успешно создает новые записи при регистрации.
* **Негативные сценарии:** * Программа успешно блокирует вход при пустых полях ввода или неверном пароле.
* Модуль регистрации корректно обрабатывает ошибки при несовпадении паролей или попытке использовать уже существующий логин.
* **Рефакторинг:** Благодаря выносу логики в отдельные `public` методы, тесты работают стабильно и независимо от графического интерфейса пользователя.
* **Стабильность БД:** Все SQL-запросы в коде соответствуют структуре таблиц, а строка подключения в `App.config` настроена верно.
